# Object-Detection-and-location-RealsenseD435

## Requirements
### Python Version
pyrealsense2.x  
Opencv-python   
numpy  
Intel Realsense SDK for D435

## How to use
When you install the essential job ,just run the main.py,the model will start to work.
```
cd Python   
python3 main.py
```
